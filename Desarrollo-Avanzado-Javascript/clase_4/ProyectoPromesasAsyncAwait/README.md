# Sistema de Reservas para Restaurante

Proyecto para practicar **Promesas** y **async/await** en JavaScript.

## Requisitos

- [Node.js](https://nodejs.org/) (v12 o superior)

## Ejecución

Abre una terminal en la carpeta del proyecto y ejecuta:

```bash
node reservaRestaurante.js
```

El script simula un sistema de reservas con 3 pruebas automáticas: reserva exitosa, mesas insuficientes, y posible fallo en el envío del correo de confirmación.
